💬 WhatsApp Clone – Frontend

A modern WhatsApp clone frontend built using React, Material UI (MUI), and Axios.
This project includes Google Login Authentication using @react-oauth/google.

🚀 Tech Stack

⚛️ React (Create React App)

🎨 Material UI (MUI)

🌐 Axios (API requests)

🔐 Google Authentication (@react-oauth/google)

🗂 React Context API (State Management)

🖥 Runs On
http://localhost:3000


Start the development server:

npm start

✨ Features

🔐 Google Login Authentication

💬 Chat UI (WhatsApp-style design)

👤 Conversation List

📩 Real-time Message Interface (UI ready)

📦 REST API integration using Axios

🎨 Responsive UI using Material UI

⚡ Clean component-based architecture

🔑 Environment Variables

Create a .env file in the root of the project (same level as package.json) and add:

REACT_APP_GOOGLE_CLIENT_ID=your_google_client_id_here


⚠️ Important:

Do NOT use quotes

Restart server after adding .env

Never upload .env to GitHub

📦 Installation

Clone the repository:

git clone https://github.com/HamzaAhmed356/whatsapp-clone-frontend-.git


Go inside the project:

cd whatsapp-clone-frontend-


Install dependencies:

npm install


Start development server:

npm start

🔐 Google Authentication

This project uses:

@react-oauth/google


Google login is handled entirely on the frontend using OAuth 2.0.
